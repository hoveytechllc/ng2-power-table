Getting started

ng2-power-table gives you the power to decide which components you want, which you don't

1. Download power-table

In order to get started download power-table from using npm:

npm install ng2-power-table

2. Configure bootloader

Modify your systemjs.config.js to load the ng2-power-table module when requested.

Basic setup

Power-Table includes several components to help manage table related functions such as paging and sorting. These components can be used selecting, except the main component TableComponent, leveraged by using the attribute ptTable.

  • ptDisplayArray: power-table outputs the results of any paging, sorting or filtering into this property.
  • ptTableState: This property provides access to any state properties of the table such as paging, sorting, filtering, etc.
