Server

1. Implement IDataPipeService and make it @Injectable

The pipe() method can ignore data: Array<any> as it is mainly relevant for client-side paging. The parameter tableState: ITableState has values ITableState.pagination.start and ITableState.pagination.pageSize that should be used for retrieving paginated data from the server.
The value ITableState.pagination.totalItemCount should be kept up-to-date. This can either be done by the pipe (e.g. count is constantly growing) or the parent component (e.g. count only changes due to user interaction with filter settings).

2. Bind an IConfiguration to the table which refers to the custom IDataPipeService

The parent component can use DefaultConfiguration.create() to get a fully functional configuration then set IConfiguration.pipeServiceType to the custom pipe.
