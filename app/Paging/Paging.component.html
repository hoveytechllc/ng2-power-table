Paging

1. Add pt-pagination element as child of ptTable.

The ptSort directive depends on ptTable main directive. The attribute value of ptSort is used to determine which property should be sorted.
The pt-pagination component does not do any of the heaving lifting for paginagion. It only has logic to display the amount of pages and the ability to navigate thru them. The heavy lifting is done by the IDataPipeService interface. This service is configurable and by default is implemented by the DefaultDataPipeService.
